Smoishele

Select multiple images from your hard drive to smoishel into an averaged face, or connect with Facebook and smoishel all the photo's you are tagged in.

Credits:

Smoishele uses the clmtrackr library for feature detection. Face bounding box detection is done using a JavaScript implementation of the ccv library. The webapp itself is maintained by Olaf Janssen.

Disclaimer:

The algorithm is slightly biased. It works best with fair-skinned men without glasses. Nothing is uploaded to the server and no personal information is collected.
